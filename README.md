## 前言
手写代码，是面试中考察候选人非常重要的手段，他能够非常的清晰的看出候选人代码编写的功底
该仓库用于快速提高代码手写能力，帮助面试者找到心仪的工作

## 如何使用

## 模板
函数说明：可以是简短的几句话，也可以是一个链接

函数实现：每一个步骤具体实现

函数测试：测试结果

参考链接：由于某些情况，需要很多的描述，可直接给一个参考链接
